# 📈 Stock Price Prediction using LSTM

This project is a simple implementation of stock price prediction using LSTM (Long Short-Term Memory) model. I built this project to understand how deep learning can be used for time series forecasting, especially in financial data.

## 🚀 What this project does

- Fetches stock data using Yahoo Finance
- Preprocesses the data using scaling
- Trains an LSTM model on historical stock prices
- Predicts future stock prices
- Visualizes actual vs predicted values

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- TensorFlow / Keras
- yFinance API

## 📂 Project Workflow

1. Import required libraries
2. Load stock data using yfinance
3. Data preprocessing (scaling and splitting)
4. Build LSTM model
5. Train the model
6. Make predictions
7. Visualize results
